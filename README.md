#  ECG-Data-Analysis-Using-Statistical-Methods-in-Python
This project focuses on the statistical analysis of electrocardiogram (ECG) data collected from 50 animals. The objective is to explore the relationship between ECG parameters and different categorical and numerical factors such as age, age type, feed type, and work type using Python.
#  Methods Used
Data cleaning and preprocessing
Descriptive statistics
Pearson correlation analysis
One-way ANOVA
Chi-Square Test of Independence
P-value interpretation and statistical significance testing
#  Tools & Libraries
Python

Pandas

NumPy

SciPy

Matplotlib / Seaborn (optional)

# Key Findings
. No significant correlation was found between age and most ECG parameters.
. Feed type showed a statistically significant effect on RR interval (p < 0.05)
. Work type did not show a significant effect on ECG heart rate.
. A significant association was identified between age type and ECG remarks using the Chi-square test.
